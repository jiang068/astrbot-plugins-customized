# astr_rela

AstrBot 人际关系管理插件

## 功能介绍

为 AstrBot 提供完整的人际关系管理功能：
- 查看好友列表、群列表
- 审批好友申请、群邀请
- 自动通过好友申请（支持关键词验证）
- 删除好友、退群
- 群事件通知（进群、退群、禁言、管理员变动等）
- 抽查群聊天记录
- 群黑名单、成员互斥管理

## 安装

克隆到插件目录，或在web控制台安装插件zip：

```bash
cd /AstrBot/data/plugins
git clone https://github.com/Zhalslar/astrbot_plugin_relationship
```

## 配置说明

在 AstrBot 面板配置：插件管理 → astr_rela → 插件配置

主要配置项：
- **管理群号**：审批信息发送目标群（留空则私发管理员）
- **最大禁言时长**：超过此时长自动退群（默认：86400秒/1天）
- **最大群容量**：限制加群数量（默认：100，设为0不限制）
- **群聊黑名单**：拒绝加入的群列表（被踢自动加入）
- **互斥成员黑名单**：检测到指定成员自动退群
- **自动抽查群消息**：群事件发生时自动抽查聊天记录
- **自动通过好友申请**：验证信息包含关键词自动通过

## 指令列表

| 指令 | 说明 |
|------|------|
| `/好友列表` | 查看所有好友 |
| `/群列表` | 查看所有群聊 |
| `/同意` (引用申请) | 同意好友申请或群邀请 |
| `/拒绝` (引用申请) | 拒绝好友申请或群邀请 |
| `/删除好友 <QQ号或@>` | 删除指定好友 |
| `/退群 <群号>` | 退出指定群聊 |
| `/抽查 <群号>` | 抽查指定群聊天记录 |
